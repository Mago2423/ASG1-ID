overview:
this project is to promote the company of chairbuy.co (a fictional company) using a website - Chairbuy.com to promote the website. this company is one that sells chairs of all sorts, there are a company that sells chairs to better the lives of our custormers. to strive to improve on thier designs and share the joy of having a chair to work with, we innovate, ideate and create chairs that carry the very essence of what a chair should be. to do this, they need user feed back and funds. thus a website that can expaned thier influence further and allow a wider audience to buy thier wares will help them greatly.
This project is a multi-page furniture e-commerce website designed to provide users with a clean and intuitive online shopping experience. The site features a homepage, product listings, an about page, and a contact form. It aims to showcase furniture products in a simple, aesthetic layout while making it easy for visitors to browse items, learn about the company, and reach out for enquiries.
The design focuses on smooth navigation, responsive layouts, and visually clear sections, ensuring that users can quickly find what they need. This project serves as both a demonstration of front-end web development skills and a functional prototype of an online retail website.

User stories:
As a visitor, I want to browse furniture products so that I can find items I am interested in buying.
As a customer, I want to view product details so that I can understand what each item offers.
As a new user, I want to know more about the company so that I can trust the brand.
As a potential buyer, I want to contact the business so that I can make enquiries or request support.
As a general user, I want a responsive layout so that the site is usable on both phones and laptops.

Existing Features
Homepage
Welcomes users with a description and provides navigation to other sections.

Products Page
Shows multiple furniture items in clean, organized boxes.
Each product has an image, a description, and associated details.

About Page
Contains the company profile and contact information.
Includes the company’s email and phone number.

Contact Page
Includes a contact form (name, email, message).
Has a submit button ready for future backend implementation.
Uses simple layout for clarity.

Navigation Bar

Responsive Layout
